# 项目-光线追踪渲染器

## 完成功能：

* 向量运算库，控制摄像机，渲染球体、平面、三角形
* 对物体添加材质，材质分三种，漫反射材质，金属镜面反射材质，透明玻璃反射材质
* 对物体添加纹理，纹理分两种，噪声纹理和图片纹理
* 抗锯齿，散焦模糊效果，添加光源
* 加载网格模型，构建BVH树
* 多线程加速，渲染天空盒
